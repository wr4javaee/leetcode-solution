# 114. 二叉树展开为链表

## 题目描述

```
给定一个二叉树，原地将它展开为链表。

例如，给定二叉树

    1
   / \
  2   5
 / \   \
3   4   6
将其展开为：

1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6
```

## 解题思路

* 前序遍历
* 遍历结果追加到左子树
* 左子树置空
